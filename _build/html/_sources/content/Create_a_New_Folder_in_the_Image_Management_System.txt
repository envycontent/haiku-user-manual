
Create a New Folder in the Image Management System
======================================================================================================

This shows you how to create a new folder in the Image Management System. 	

New Folder
-------------------------------------------------------------------------------------------

.. image:: images/Create_a_New_Folder_in_the_Image_Management_System/media_1354549862656.png
   :align: center
   

Click on New Folder.


Name the new folder
-------------------------------------------------------------------------------------------

.. image:: images/Create_a_New_Folder_in_the_Image_Management_System/media_1354549991650.png
   :align: center
   

1. Enter the folder name.
2. Enter a description of the content of the folder.
3. Click on Change. This takes you to the following page where you can specify where you would like to create your folder. 


Choose the folder location
-------------------------------------------------------------------------------------------

.. image:: images/Create_a_New_Folder_in_the_Image_Management_System/media_1354550483414.png
   :align: center
   

1. Click on the folder where you would like to add the new folder.
2. Click on OK.


Create the folder
-------------------------------------------------------------------------------------------

.. image:: images/Create_a_New_Folder_in_the_Image_Management_System/media_1354550646247.png
   :align: center
   

Click on Create.



.. image:: images/Create_a_New_Folder_in_the_Image_Management_System/media_1354550801919.png
   :align: center
   

1. Click on Finish to create the folder, or 
2. Click on Upload files if you want to create the folder and add files to the folder.
